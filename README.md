# doc 知识分享整理文档
该项目主要用于搜集整理总结学习文档，为了方便查阅和归类需要按照规范来编写目录


## 规范
- 项目主要根据前端涉及范围氛围多个一级目录，可根据文档内容自行添加目录,目录内的readme.md内可做目录具体描述
- 单个md文件可以直接放在一级目录下，对应图片放在一级目录下的images文件夹内
- 分享内容整理是一个文件夹(包含多个md文件和代码脚本)，则直接在一级目录下添加子目录，并包含自己的images文件夹存放图片
